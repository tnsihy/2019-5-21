`某计算机字长32位，它的存储容量为256KB，按半字编译，它的寻址范围是`  

最终结果：  
按半字编译的寻址范围是`128KB`

1字节(B) = 8位  
256KB = 256 * 1024 * 8位  
若按半字编译，即半字的位数是16位  
则范围是 256 * 1024 * 8位 / 16位 = 128 * 1024B = 128KB
若按单字编译，即32位  
则范围是  256 * 1024 * 8位 / 32位 = 64 * 1024B = 64KB
